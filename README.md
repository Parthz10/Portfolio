# 🖥️ Windows 11 Inspired Portfolio

A highly interactive, immersive portfolio built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**, designed to look and feel exactly like the Windows 11 desktop environment.

![Windows 11 Portfolio](public/wallpaper.jpg)

## ✨ Features

- **Windows 11 Aesthetic**: Authentic taskbar, start menu, system tray, and glassmorphism effects.
- **Dynamic Windowing System**: Multi-window support with dragging, resizing, focus management, and minimizing/maximizing.
- **Immersive OS Experience**: Includes a boot sequence, login screen, and right-click context menus.
- **Portfolio Sections**:
  - 🏠 **Home**: Quick overview and stats.
  - 📂 **Portfolio**: Projects and showcase.
  - 🎓 **Education**: Academic background.
  - 💼 **Experience**: Professional journey.
  - 🏆 **Certifications**: Gallery of achievements.
  - 📊 **Skills**: Interactive skill charts.
  - ✉️ **Contact**: Functional contact form.
- **Search & Navigation**: Fully functional centered search bar in the taskbar.

## 🚀 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Components**: [Sonner](https://sonner.emilkowal.ski/), [Recharts](https://recharts.org/), [React Hook Form](https://react-hook-form.com/)

## 🛠️ Getting Started

### Prerequisites

- Node.js 18+
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Parthz10/Portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📂 Project Structure

- `/app`: Next.js App Router pages and API routes.
- `/components/os`: Core OS components (Taskbar, StartMenu, Window system).
- `/components/portfolio`: Individual portfolio sections.
- `/lib`: State management, data stores, and utility functions.
- `/public`: Static assets, wallpapers, and documents.
